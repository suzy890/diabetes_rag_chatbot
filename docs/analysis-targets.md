# analysis-targets.md — 최소 결과물(분석 목표) 정의 · 역순 접근

> **문서 버전:** v0.1 · **작성 기준일:** 2026-07-09
> 연구팀 제안(역순 접근)을 반영: **"실증 환경부터 설정하고, 최소한의 결과를 어떻게 보여줄지부터 정한 뒤, 거기서 결정사항을 도출"**한다.
> 즉 화면·기능이 아니라 **최종적으로 보고 싶은 결과 → 필요한 데이터 → 확정해야 할 결정** 순으로 거꾸로 내려간다.

---

## 1. 이번 실증에서 "최소한으로 보여줄 결과"

강한 인과("시스템이 행동을 바꿨다") 대신, **중재 노출과 반응의 관계·사용 전후 변화·현장 수용성**을 본다([PRD.md](PRD.md)).

### A. 기술(descriptive) 최소 결과
- 참여자별 **접속 횟수·세션 길이**
- **넛지 노출량 vs 반응량** (예정→노출→응답 전환)
- **행동 약속 → 실제 수행 전환율/소요시간** (의도≠수행)
- **질문 수·주제**, **출처 클릭률**
- **안전 응답 발생 빈도·유형** (과다 트리거 점검 포함)
- (설문이 있으면) **시점별 측정값 변화** (신뢰·이해도·자기효능감·행동의도 등)

### B. 관계(탐색) 최소 결과
- 넛지 반응률과 행동의도의 관계
- 출처 클릭 경험과 이해도/신뢰의 관계
- 사용빈도와 자기효능감의 관계
- (종단) 시간에 따른 넛지 반응률·질문 빈도 변화

## 2. 각 결과에 필요한 데이터 (이미 스키마에 있는지 점검)

| 결과 | 필요 데이터 | 스키마 위치 | 준비됨? |
|------|------------|-------------|---------|
| 접속·세션 | sessions(started/ended) | `sessions` | ✅ |
| 넛지 전환 | 예정·노출·응답 | `nudge_events` (scheduled/displayed/responded) | ✅ |
| 의도→수행 | 약속·수행 | `nudge_events.action_commitment` + `action_followups.completion_status` | ✅ |
| 질문·출처 | 질문·답변·출처클릭 | `messages`, `events(source_clicked)` | ✅ |
| 검색 품질 | 검색 근거 | `retrieval_logs` | ✅ |
| 안전 발생 | 안전 이벤트 | `events(safety_message_shown)` | ✅ |
| 설문 변화 | 시점별 측정 | `measurements(measurement_timepoint)` | ✅ 구조는 있음, **문항·시점 미정(U8)** |
| 비용 | 토큰·비용 | `model_calls` | ✅ |

> **결론:** 결과물을 만들기 위한 **데이터 구조는 이미 마련**되어 있다(Day 3). 다만 아래 결정이 있어야 실제 값을 채운다.

## 3. 결과물로부터 도출되는 "먼저 정할 결정" (우선순위)

역순으로 보면, 최소 결과물을 만들려면 아래 순서로 결정이 필요하다.

1. **U8 설문·측정시점** — B(관계 결과)와 사용 전후 변화를 보려면 baseline/사후 시점과 문항이 필요. → 결과물에 직결, **우선순위 높음**.
2. **U1 참여자 인증 방식** — 참여자별 데이터를 묶는 출발점. Phase 1 첫 작업 전 필요.
3. **U5 넛지 시작 영역 / U6 횟수·간격** — 넛지 전환 결과의 조건.
4. **U2~U4 문서·청킹·임계값** — 질문·검색 품질 결과의 조건(Phase 2).
5. **U11 배포환경 / U10 IRB / U12 안전 문구** — 실증 환경·윤리 전제.

> 이 우선순위는 **결과물 기준**이다. "이 결과를 보려면 이 결정이 필요하다"가 명확한 것부터 정한다.
> 미결정 항목 전체 목록·이유는 [DECISIONS.md](DECISIONS.md) 참고.

## 4. 실증 최소 산출 형태

- 연구자가 **CSV로 내보내** 위 A/B 결과를 재구성할 수 있으면 MVP 데이터 목표 달성([PRD.md](PRD.md) 완료 기준).
- 분석 자체(통계)는 이번 MVP 필수 아님. **데이터가 빠짐없이 쌓이는 것**이 1차 목표(수집 우선 원칙, D11).
