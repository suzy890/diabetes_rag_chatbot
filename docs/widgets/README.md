# Notion 인터랙티브 위젯 원본

노션 개발 로그에 삽입한 인터랙티브 HTML 블록의 **원본 파일**입니다.
(노션의 파일 업로드형 HTML 블록은 노션 안에서 코드를 직접 수정할 수 없어서, 원본을 여기서 관리합니다.)

| 파일 | 용도 | 삽입된 노션 페이지 |
|------|------|------------------|
| `roadmap-widget.html` | Phase 0~5 인터랙티브 개발 로드맵 | 개발 로그 · Day 1 |
| `cost-calculator.html` | API 비용 계산기 | 개발 로그 · Day 1 |

## 문구/내용을 수정하려면

1. VSCode에서 해당 `.html` 파일을 엽니다.
2. 고칠 글자를 찾아 수정합니다.
   - **로드맵 단계 이름·할 일:** 파일 안 `PHASES` 목록의 `t`(제목), `tasks`(할 일) 값을 고칩니다.
   - **계산기 라벨·기본값:** `<label>` 텍스트나 `value="..."` 숫자를 고칩니다.
3. **미리보기:** 파일을 웹브라우저로 열면 실제 모습과 동작을 확인할 수 있습니다.
   (VSCode 마크다운 미리보기로는 동작이 안 보입니다 — 브라우저로 여세요.)
4. 수정이 끝나면 Claude에게 "이 위젯 노션에 다시 올려줘"라고 하면, 새 버전으로 교체해 드립니다.

> ⚠️ 외부 인터넷 통신(폰트·이미지·CDN)은 노션 샌드박스에서 차단됩니다.
> CSS·JS는 반드시 파일 안에 그대로 두세요(외부 링크 추가 금지).
