# medical-safety.md — 의료·건강정보 안전 규칙

> **문서 버전:** v0.1 · **작성 기준일:** 2026-07-08
> [CLAUDE.md](../../CLAUDE.md)의 하위 세부 규칙. 고령 당뇨 환자를 다루므로 **안전이 정확성보다 우선**한다.
> 구체적 판별 기준·고정 안내문 문구는 이후 `SAFETY_RULES.md`에서 확정하며, **문구는 의료전문가 검토를 받는다.**

---

## 1. 절대 하지 않는 것

- **진단을 제공하지 않는다.** (예: "그 증상은 ○○입니다", "합병증입니다" 같은 개인별 진단)
- **처방을 제공하지 않는다.** (특정 약·용량·치료를 개인에게 지시하지 않는다)
- **복약 변경을 지시하지 않는다.** (약물 용량 증감·중단·추가 복용을 직접 지시하지 않는다)
- 근거 없는 **완치·치료 효과를 주장하지 않는다.**
- 위해 가능성이 있는 행동을 권하지 않는다.

> 이 항목들은 일반적 정보 제공과 의료진 상담 권고로 대체한다. (예: "일반적으로 …라고 알려져 있습니다. 구체적인 약 조절은 담당 의료진·약사와 상의하세요.")

## 2. 근거 부족 시 답변 보류

- 관련 문서 청크가 없거나 검색 점수가 낮으면 **추측해서 답하지 않는다.**
- 대신 "현재 지식베이스에서 충분한 근거를 찾지 못했다"고 안내한다.
- RAG는 할루시네이션을 **줄이는** 장치일 뿐 완전히 제거하지 못한다. 근거 부족 보류는 그 안전장치의 핵심이다.

## 3. 질문 분류와 처리

| 질문 유형 | 처리 |
|-----------|------|
| **일반 정보 질문** | 승인 문서를 검색해 일반적 자기관리 정보를 제공 + 출처 표시 |
| **근거 부족 질문** | 추측 금지, 근거 부족 안내 |
| **의료진 확인 필요 질문** | 일반 정보 + 의료진·약사 상담 권고 |
| **진단·처방·복약 변경 질문** | 직접 지시 금지, 일반 정보 + 전문가 상담 권고 |
| **응급·고위험 질문** | 일반 RAG 흐름보다 **고정된 안전 안내를 우선** |
| **당뇨와 무관한 질문** | 범위 밖임을 안내 |

## 4. 응급·고위험 상황

- 응급 가능성이 있는 증상, 고위험 키워드가 감지되면 **일반 답변보다 고정 안전 안내문을 먼저** 보여준다.
- 어떤 키워드·상황을 고위험으로 볼지, 어떤 문구를 쓸지는 `SAFETY_RULES.md`에서 정하며 **의료전문가 검토가 필요**하다. (임의로 문구를 생성하지 않는다)

## 5. 의료진·약사 확인 권고 기준

다음의 경우 전문가 상담을 권고한다.

- 개인별 약물 용량·복약 일정에 관한 질문
- 개인별 진단·합병증 판단이 필요한 질문
- 증상이 지속·악화되거나 응급 가능성이 있는 경우
- 근거 문서 범위를 벗어나 개인 맞춤 판단이 필요한 경우

## 6. 안전성 로그

- 안전 제한 응답이 작동하면 **`safety_message_shown` 이벤트와 질문 범주를 기록**해 사후 검토할 수 있게 한다.
- 이 로그는 고위험 질문의 발생 빈도·유형을 파악하고 안내문을 개선하는 근거가 된다.

## 7. 넛지와 안전의 경계

- **복약 넛지는 "복약하셨어요?" 수준의 행동 상기까지만** 한다. 용량·종류 등 의료적 판단으로 확장하지 않는다.
- 넛지 문구는 명령·공포·죄책감·강압 표현을 쓰지 않고, 거절/나중에 선택지를 제공한다. (세부는 `NUDGE_RULES.md`)
